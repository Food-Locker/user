# 백엔드 서버 실행 가이드

## 서버 실행 방법

1. **서버 폴더로 이동**:
   ```bash
   cd server
   ```

2. **의존성 설치** (처음 한 번만):
   ```bash
   npm install
   ```

3. **서버 실행**:
   ```bash
   npm start
   ```

   또는

   ```bash
   node index.js
   ```

4. **서버가 정상적으로 실행되면**:
   ```
   서버가 포트 3001에서 실행 중입니다.
   MongoDB 연결 성공
   ```

## 문제 해결

### 404 에러가 발생하는 경우
- 백엔드 서버가 실행되지 않았을 가능성이 높습니다.
- 위의 단계에 따라 서버를 실행해주세요.

### MongoDB 연결 오류
- `.env` 파일에 `MONGODB_URI`가 올바르게 설정되었는지 확인하세요.
- MongoDB Atlas에서 IP 화이트리스트 설정을 확인하세요.

### 포트가 이미 사용 중인 경우
- 다른 터미널에서 서버를 실행 중일 수 있습니다.
- `Ctrl + C`로 기존 서버를 종료하거나, 다른 포트를 사용하세요.
